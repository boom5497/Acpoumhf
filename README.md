# 项目简介

## 核心功能
- 咨询功能集成百度千帆大模型
- Echarts可视化统计
- 首页推荐采用协同过滤算法
- 用户功能：登录模块、注册模块、图书馆模块、图书留言反馈模块、图书馆推荐、首页模块、在线客服模块
- 管理员功能：图书分类管理模块、个人信息模块、用户管理模块、轮播图管理模块、图书馆管理、留言反馈管理模块、预约记录管理模块、预约取消管理模块、在线客服管理模块
- 游客功能

## 适用场景
- 图书馆信息管理
- 在线图书推荐
- 图书借阅与归还管理
- 用户互动交流

## 技术特点
- B/S架构
- MVC模式
- Web框架开发
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
